## 简介

Launcher Theme Enhancer 是一款 SJMCL 扩展，将此前三个独立插件整合为一个无缝体验，同时解决了三者同时开启时的冲突问题。

内置中英文切换按钮，可在设置面板中一键切换语言，无需安装不同语言版本。

支持与 sjmcl-video-background、sjmcl-weather-effects、sjmcl-city-weather 等旧版独立插件共存，但建议卸载旧插件后使用本整合版以获得最佳体验。

## 功能特色

### 🌤 城市天气（增强版）
- 通过 Open-Meteo API 获取实时天气数据
- 覆盖全球 200+ 城市
- 天气匹配的粒子背景与氛围色调
- 相比旧版，天气响应更精准、过渡更平滑

### 🎬 动态视频背景
- 上传 MP4/WebM 视频文件作为启动器背景
- 修复了旧版动态背景会覆盖其他插件页面的问题
- 音量控制、播放/暂停/删除

### ✨ 粒子特效

**新增粒子效果：**
- 附魔 — 浮动的附魔符文
- 幽匿 — 循声守卫风格的幽匿粒子
- 冻结 — 冰雪霜冻粒子
- 代码 — 矩阵风格的代码雨
- 灰尘 — 空气中漂浮的尘埃
- 活力 — 充满生机的光点

**重写优化：**
- 火焰、雪、日落、水下、极光 — 全面重写，效果更流畅

**已移除：**
- Minecraft 风格粒子（因整合重构而移除）

### 🌐 语言切换
- 设置面板中一键切换中英文
- 不再需要下载不同语言版本

> ⚠️ **注意：已移除粒子和动态视频的记忆功能**，每次打开启动器需重新设置。对于此改动深表歉意，由于技术力的原因，以后的更新很可能会考虑加回。

## 使用方法

1. 在 Releases 下载 org.yoshino.launcher-theme-enhancer-1.0.0.sjmclx
2. 打开 SJMCL → 设置 → 扩展管理 → 导入
3. 选择文件并启用即可

## 兼容性

- 最低启动器版本: 1.1.3
